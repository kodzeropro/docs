# Что такое коллекции

Коллекция — это структурированное хранилище данных в Kodzero. По сути, это таблица в базе данных с автоматически сгенерированным REST API.

## Основные компоненты

Каждая коллекция состоит из:

| Компонент | Описание |
|-----------|----------|
| [**Данные**](./data) | Записи (строки) в таблице |
| [**Схема**](./schema) | Структура полей и их типы |
| [**Настройки API**](./api-settings) | REST-эндпоинты, права доступа и конфигурация методов |

## Создание коллекции

1. Перейдите в раздел **«Коллекции»** в панели управления
2. Нажмите **«Новая коллекция»**
3. Введите название коллекции
4. Настройте схему данных
5. Сохраните изменения

::: tip :pushpin: Рекомендация по названию
Используйте понятные названия в нижнем регистре: `products`, `users`, `orders`. Это упростит работу с API.
:::
